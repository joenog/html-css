Formatação de textos

Texto strong - destaque

Para destacarmos um texto semanticamente usamos a tag strong

Texto em - enfase

Para darmos enfase ao texto usamos a tag em

Texto marcado - Mark

Para marcamos um texto(marcador) usamos a tag mark. A cor da marcaçao pode ser editada na CSS

Texto grande e pequeno - Big, Small

Para textos grandes usamos a tag big

Para textos pequenos usamos a tag small

Texto deletado - Del

Para texto deletado ou riscado usamos a tag del

Texto inserido, sublinhado - INS

Para texto inserido/sublinhado usamos a tag ins
